# SAE-J1939 Library

Provides helper functionality conforming to the SAE-J1939 specification.

## Usage

Add an entry to your `Cargo.toml`:

```toml
[dependencies]
sae-j1939 = "0.1.1"
```

## References

- [SAE J1939 Standards Collection](https://www.sae.org/publications/collections/content/j1939_dl/)
